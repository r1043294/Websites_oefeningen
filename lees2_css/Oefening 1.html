Vrije software

Vrije software is de verzamelnaam van software die tegenwoordig vooral verspreid wordt onder de
GNU General Public License (GPL).

Open souce

Open source is een tem die eind jaren negentig is bedacht om gebruik van spftware door berijvem te stimuleren.
Door sterk de nadruk tenleggen op rechten van gebruikers en het vrijgeven van zoveel mogelijk broncode, hadden veel
bedrijven de indruk dwt zij geen gebruik konde makne van vrije softwre.

Misvattingen

Vrije softwre mag wel verkocht worden. Er bestaat dus, in principe althans, wel degelijke commerciele vrije software. Al is
dat wel heel moeilijk vol te houden: elke koper van zulke vrije software mag immers zelf de software beginnen te kopieren
en verder verhandele, wat natuurlijk tot een neerwaartse druk op de prijs leidt.
